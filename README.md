# YouTube Homepage Clone

This is a basic YouTube homepage UI clone built using **HTML and CSS**.

The project focuses on practicing front-end fundamentals such as layout, styling, and structuring a webpage similar to the YouTube homepage.

## Tech Stack
- HTML5
- CSS3

## Features
- Basic YouTube homepage layout
- Clean UI design
- Built using only HTML and CSS

## Purpose
This project was created for learning and practice purposes to improve front-end development skills.
